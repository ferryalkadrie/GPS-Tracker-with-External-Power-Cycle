# SIM800L GSM Module

## Overview

The SIM800L is responsible for GSM/GPRS communication and internet connectivity.

It uploads GPS coordinates collected by the ESP32 to the ThingSpeak cloud platform.

---

# Features

- Quad-band GSM/GPRS
- AT Command Interface
- TCP/IP Stack
- HTTP Communication
- Low Power Operation

---

# ESP32 Connection

| SIM800L | ESP32 |
|----------|-------|
| TXD | GPIO16 |
| RXD | GPIO17 |
| RST | GPIO25 |
| VCC | External Power Supply |
| GND | Common Ground |

---

# Communication

The ESP32 communicates with the SIM800L using UART1 at 9600 baud.

---

# AT Command Sequence

1. Check modem
2. Configure APN
3. Enable GPRS
4. Connect TCP
5. Send HTTP Request
6. Close Connection

---

# Error Recovery

If the modem becomes unresponsive, the ESP32 performs a hardware reset through GPIO25 before retrying communication.
