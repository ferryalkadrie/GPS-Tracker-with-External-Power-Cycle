# Troubleshooting

## ESP32 Does Not Boot

Possible causes:

- Power supply problem
- AO3400 wiring issue
- ESP-01 power cycle active

---

## GPS No Fix

Possible causes:

- Poor antenna placement
- Indoor environment
- Insufficient satellite visibility

---

## SIM800L Cannot Connect

Check:

- SIM card
- APN configuration
- GSM signal strength
- Power supply stability

---

## ThingSpeak Upload Failed

Verify:

- Internet connection
- API Key
- TCP connection
- HTTP request format

---

## ESP-01 Does Not Wake

Verify:

- GPIO16 connected to RST
- Deep Sleep configuration
- Stable 3.3V supply

---

## ESP32 Random Restart

Possible causes:

- Brownout
- Weak power supply
- GSM current spikes
